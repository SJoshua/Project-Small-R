# Project-Small-R
A Telegram Bot Framework in Lua.

## Requirement
* Operating System: UNIX
* Lua 5.1+

## Dependence
* Lua-socket
* Lua-cjson
* Lua-ssl
* Lua-ltn12
* Lua-multipart-post

## Bots
* [@small_robot](https://t.me/small_robot): A chat bot with some interesting features in English, Chinese and Japanese.
